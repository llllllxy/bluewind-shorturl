蓝风短链平台 主页 API接口 租户登录 注册/登陆

API说明(key:登录后查看)

1 请求地址

http://create.link66.cn/api

2 请求方式

GET/POST

3 输入参数

参数名 参数类型 是否必需 描述 示例值
longUrl String 是 urlencode('链接') http%3a%2f%2fwww.baidu.com
format String 否 调用格式，值为json或jsonp或不传，不传时返回数据为text  
callback String 否 回调函数名，当format=jsonp时必传  
key String 是 登录后查看  
expireDate String 否 如,expireDate=2020-12-31  

4 返回结果

参数名 参数类型 描述 示例值
url String 生成的短网址，如果生成失败，则返回空 http://3z6.cn/2AuiCe
err String 错误描述 生成成功时该值为空

4-1 【TXT格式】请求与返回示例

请求：
http://create.link66.cn/api?longUrl=urlencode('http://www.baidu.com')&key=asdk1871as&expireDate=2020-12-31

返回：
http://3z6.cn/2AuiCe

4-2 【JSON格式】请求与返回示例

请求：
http://create.link66.cn/api?format=json&longUrl=urlencode('http://www.baidu.com')&key=asdk1871as&expireDate=2020-12-31

返回：
{"url":"http://3z6.cn/2AuiCe","err":""}

4-3 【JSONP格式】请求与返回示例

请求：
http://create.link66.cn/api?format=jsonp&longUrl=urlencode('http://www.baidu.com')&callback=callbackname&key=asdk1871as&expireDate=2020-12-31

返回：
callbackname({"url":"http://3z6.cn/2AuiCe","err":""})
